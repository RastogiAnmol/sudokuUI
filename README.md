## Sudoku
### Building locally

## I have modified sudoku game present on internet to use backend swagger API
Prerequisites:

- NodeJS
- NPM
- [Webpack][webpack] (installed globally)
- [LESS][less] and the [clean CSS plugin][lesscleancss] (installed globally)

```bash
$ git clone https://github.com/andreynering/sudoku.git
$ cd sudoku
$ npm install
$ npm run buildjs
$ npm run buildcss
```
